# StormShop

StormShop - проект содержащий в себе серверную и клиентскую часть.
Для запуска приложения необходимо:
- Скачать java 8;
- Clean instaill - осуществляется через инструмент для сбори java проекта - maven;
- Запустить серверную за тем клиентскую часть;
Адрес для подключения к к серверу - http://localhost:4567 (указывается при запуске клиентской части);

Файл с содержимом товаров в магазине можно посмотреть в корневой папке прокта JavaSparkBackend с наименованием goods.json
В прокте используются следующие фреймворки:
- spark;
- jackson;
- apache;
