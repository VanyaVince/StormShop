# StormShop
StormShop - проект содержащий в себе серверную(java) и клиентскую часть(javaFX), для платформ JDK.

Особенности:
- Написан под Desktop приложения.
- В проекте используется фреймворк Spark: все запросы осуществляются по протоколу - http.

В прокте используются следующие библиотеки:
- jackson - сериализация и десериализация объектов из json файла и обратно;
- apache - хеширование пароля пользователя, на основании алгоритма SHA1;

Для запуска приложения необходимо:
- Скачать java 8;
- Clean instaill - осуществляется через инструмент для сбори java проекта - maven;

Фунционал программы: 
- Запустить серверную за тем клиентскую часть;

При старте приложения необходимо осуществить подключение к серверу:

Адрес для подключения к серверу - http://localhost:4567 (указывается при запуске клиентской части, "Прописан изначально");

Приложение осуществляет следующую логику: приобретение, добавление товаров в магазин;

Для добавления товаров необходимо иметь права администратора, пройдя авторизацию;
